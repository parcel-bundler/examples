## React

Clone repository and go to `react` folder:

```bash
$ git clone https://github.com/parcel-bundler/examples
$ cd examples/react
```

Install dependencies and run dev server:

```bash
$ npm install
$ npm run start
```

## Vue.js

Clone repository and go to `vuejs` folder:

```bash
$ git clone https://github.com/parcel-bundler/examples
$ cd examples/vuejs
```

Install dependencies and run dev server:

```bash
$ npm install
$ npm run start
```
